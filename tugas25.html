mengenal semantic elements di HTML5

  • article

    • ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan gunakan dengan element header dan footer. karena tag article memmiliki type block, artinya element ini memmiliki lebar sesuai browser jika kita menentukan lebarnya

  • aside

    • Element ini bisa kita gunakan untuk melengkapai layout utama. Misal kita meiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identititas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya
